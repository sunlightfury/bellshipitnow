    Installation

    Once you have downloaded the application, install the application on your server and sign in to your dashboard.

    Server Requirements

    • VPS server
    • Apache with PHP and MySQL server
    • PHP v5.6
    • MySql
    • Openssl Enabled

    Quick Installation Guide

    • Unzip the Deprixa pro - integrate web system.zip which you have downloaded from the Envato Market.
    • Upload your downloaded zip file to your sever using any FTP client or cpanel.
    • Create a MySql Database via your web hosting providers Control Panel (e.g. cPanel). Once you have created the database, remember the following information:
      • Database Name
      • Database Username
      • Database Password
      • Database Host
    • Open your web browser and enter your url (http://yourdomain.com) or if files is in your sub directory enter your url with directory name like http://yourdomain.com/foldername in the address bar.
    • If everything is fine, you will redirect to installation page.
    • Follow the instructions and click on the Next button. Then enter the database details which you have collected earlier and press next and enter your name, email and password which will be required to login in the application. Then click on the Finish button and wait a while. The system will create necessary tables in database, files and configuration and redirect to you on the system login page.
    • That's all! After installation delete the installation folder lcated in installation directory.

    Create shipments

    You can create or manage local or international shipments here.

    List of Create Shipment Section Features

    1. Who should send the package before going to manage the shipments
    2. In this session are the data of the recipient.
    3. It is important to place a cell phone number that works and can receive a text message about the shipment.
    4. This session is the information of the shipment, here you will find, name of office, name of the driver, type of package, name of messaging, type of payments, time of shipment and date of collection of the package.
    5. This session works to calculate your shipment, there is shown tax and customs charges, plus details of the shipment as calculation of weight and real weight of the package.

    Create container

    This module is used for maritime or land shipping companies here.

    List of Create Container Section Features

    1. Who should send the package before going to manage the shipments
    2. This session we can choose the data for the shipment of container either terrestrial or maritime, besides choosing the terms of business and type of currency for the transaction.
    3. This session we can add the dimensions, net weight and gross weight for the shipment of the container.
    4. This session we can add customs tax or tax, in addition to the container detail or shipping terms.
    5. This session we can add the items or products that go in the container, you can add the ones that you need, in the end the system calculates the quantity and the total price.

    Create consolidated

    In these modules only 3 steps can consolidate your packages in a single bag here.

    List of Create Container Section Features

    1. In this session you upload bulk mailings as a database.
    2. In this session download the excel example as a csv database and follow the instructions in each line to avoid errors when uploading the file.
    3. In this important session you can edit all your packages to consolidate, adding line to line and massively save your packages.
    4. In this session are the packages that will be consolidated.
    5. In this button we go to the next module where the recipient is chosen who receives the consolidated.
    6. In this button we can delete all the uploaded or edited data if you are not sure of the packages to be consolidated.
    7. In this session you look for the shipping numbers of each package and you enter and it is added automatically to the table
    8. In this session the data of the recipient and weight of the shipment and quantity of packages are shown
    9. In this session the consolidated packages are already shown. at least it must be more than two so that it can be consolidated
    10. In this button we open the modal window to finish filling out the consolidated shipping form
    11. In this session the shipping data is assigned and the driver is assigned who has the consolidated
    12. In this session the consolidated price is calculated, the data are, all these data are configurable:
      • pound price
      • additional pound
      • Shipping handling
      • Fuel management
      • Shipping re expedition
      • Logistics for each tracking
      • Freight value surcharges
      • Safe total declared value
      • Nationalization of the shipment
      • Customs tariffs
      • VAT

    Shipping Rates

    In this module you can set the shipping rates according to the weight here

    1. In this button you can add rates according to your weight and shipping price locally or internationally.
    2. In this area the data that was configured so that the client from the web can consult the weight is displayed and automatically the system searches for the desired range and according to the configuration shows the price of that shipment.

      The system calculates the weight and according to the configuration that you make of your weights according to the tariffs according to the weight, the formula looks for ranges of weights.

    Settings Company

    In this area you can configure your system according to your needs here.

    List of System Settings Features

    1. System Configuration: In this session, basic information about the company is recorded, such as name, address, telephone number, country, city, post code, site URL and e-mail address.
    2. Authorization of records: In this session we choose the data that authorize the registration of forms from the web as: Registration Verification, Auto Registration, Allow Registration, Registration Notification.
    3. Logo and Favicon: In this session the logo and the icon called favicon are changed.
    4. Payment method: In this session you configure the payment method by paypal, this payment tools have payments directly paypal, you can also pay with mastercard credit cards, visa and many more..
    5. Time zone and Traslate: In this session choose your time zone and language as Spanish, English, French, Italian and Russian.
    6. Default Mailer: In this session you can choose your mail platform as PHP mailer or SMTP, the important thing is to know how to configure your correct data so that the system notifies the client or administrator of the processes performed in the system.

    Tracking, Taxes and fees

    1. Tracking Shipping: In this session you configure the letters and digits of the shipment that is generated automatically or manually from the shipping form.
    2. Tracking Container: In this session you configure the letters and digits of the container shipment.
    3. Tracking Consolidated: In this session the consolidated letters and digits of the shipment are configured.
    4. Terms invoice: In this session you configure the terms of the invoice, the signature of the client and the company.
    5. Taxes and fees Shipping: In this session you configure the data or formulas to calculate the shipment.
    6. Taxes and fees Consolidated: In this session you configure the data or formulas to calculate the consolidated shipment, it includes several items.

    Settings SMS

    • SMS Twilio: In this session you must create a user in TWILIO text message messaging, there purchases credit so that the messages can reach the desired person, important to activate TWILIO if you want to use it.
    • SMS Nexmo: In this session you must create a user in NEXMO text message messaging, there purchases credit so that the messages can reach the desired person, important to activate NEXMO if you want to use it.
